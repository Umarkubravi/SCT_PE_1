# Task 01 - Writing Better Prompts

## Objective
Improve vague prompts into clear and specific prompts to get better AI responses.

---

## Example 1

### Vague Prompt
Explain climate change.

### Improved Prompt
Explain climate change in 3 simple bullet points for a 12-year-old student.

### Why It Is Better
- Specifies format
- Defines audience
- Makes response clearer

---

## Example 2

### Vague Prompt
Write about Python.

### Improved Prompt
Write a short introduction to Python programming for beginners in under 150 words.

### Why It Is Better
- Defines target audience
- Adds word limit
- Makes output focused

---

## Example 3

### Vague Prompt
Tell me about fitness.

### Improved Prompt
Create a 7-day beginner fitness plan for college students without gym equipment.

### Why It Is Better
- Gives context
- Specifies audience
- Defines requirements clearly

---

## Conclusion

Clear and specific prompts generate more accurate and useful AI responses. Prompt engineering improves communication with AI systems.
